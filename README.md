# 🏫 AutoCheckIn / 不上早八 跨平台学习通签到app

**学习通签到客户端，2025 可用**  

主要解决 **纯血鸿蒙** 设备没有原生签到软件的问题。请勿用于代签等行为  
基于 **Flutter** 开发，可自行编译为 **Android、iOS、macOS、Windows**，甚至纯血鸿蒙版本。  
当前支持扫码签到，普通签到，位置签到，后续会更新**任意位置位置签到**，**好友联动签到**等功能
与学习通完全独立，不会顶号，不显示同一设备不允许重复签到


---

## 🚀 功能特点

- ✅ 登录 / 登出  
- ✅ 课程加载  
- ✅ 扫码签到（支持滑块验证）
- ✅ 位置签到（支持滑块验证）
- ✅ 普通签到（支持滑块验证）
- 🔜 后续增加联动签到功能
